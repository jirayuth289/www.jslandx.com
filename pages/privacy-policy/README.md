## 3. ข้อมูลที่เราอาจเก็บ

### 3.1 ข้อมูลที่คุณให้โดยตรง

- ข้อมูลบัญชีผู้ใช้ (เช่น อีเมล ชื่อผู้ใช้) หากมีระบบ login
- เนื้อหาที่คุณสร้างหรืออัปโหลด เช่น สคริปต์ คำบรรยาย (caption) วิดีโอ ภาพปก
- การตั้งค่าการเผยแพร่ เช่น แพลตฟอร์มเป้าหมาย กำหนดเวลาโพสต์ ลิงก์สินค้า (product link)
- ข้อความติดต่อเมื่อคุณสื่อสารกับเรา

### 3.2 ข้อมูลจากบัญชีโซเชียลมีเดีย (ผ่าน OAuth)

เมื่อคุณเชื่อมต่อบัญชี TikTok หรือแพลตฟอร์มอื่น เราอาจได้รับข้อมูลจากแพลตฟอร์มนั้นผ่าน Outstand เช่น:

- รหัสผู้ใช้ (user ID) / ชื่อที่แสดง (display name)
- ชื่อผู้ใช้ (username)
- รูปโปรไฟล์
- สถานะการเชื่อมต่อบัญชี
- สิทธิ์ (scopes) ที่คุณอนุญาต เช่น การโพสต์วิดีโอ

**เราไม่เก็บรหัสผ่าน TikTok ของคุณ** การยืนยันตัวตนทำผ่าน OAuth ของ TikTok และผู้ให้บริการที่ได้รับอนุญาต

### 3.3 ข้อมูลที่เกิดจากการใช้งาน

- บันทึกการใช้งาน (log) เช่น เวลาเรียก API สถานะงานโพสต์ ข้อผิดพลาด
- ข้อมูลอุปกรณ์และเบราว์เซอร์ (IP address, user agent) ใน log ของเซิร์ฟเวอร์
- ข้อมูลการใช้ AI (หากใช้) เช่น prompt ที่ส่งไปยังผู้ให้บริการ AI — ตามนโยบายของผู้ให้บริการนั้น

### 3.4 ข้อมูลที่เราไม่ตั้งใจเก็บ

เราไม่ตั้งใจเก็บข้อมูลที่ละเอียดอ่อนพิเศษ (เช่น ข้อมูลสุขภาพ ศาสนา ข้อมูลชีวภาพ) เว้นแต่คุณใส่ในเนื้อหาโดยตรง
